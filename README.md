# indian-train-running-status-api
Flask API that converts Indian train running status from runningstatus.in into structured JSON including current station, arrival/departure times, delay, and platform details.
## Tech Stack

- Python  
- Flask  
- BeautifulSoup  
- Requests  

## Disclaimer

This project is an independent educational and development tool.

This API scrapes publicly available information from:  
https://runningstatus.in

This project is NOT affiliated with:

- Indian Railways  
- runningstatus.in  
- IRCTC  

Data accuracy depends on the source website.

This repository does not store or redistribute proprietary data.

Intended for personal, research, and learning purposes only.

If the website owner requests removal, the project will be taken down.
